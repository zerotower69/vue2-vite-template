# install first

`npm install`  or `yarn`

# dev

`npm run dev` or `yarn dev`

# build

`npm run build` or `yarn build`

# plugins

## sass

if you want to use sass, like this:

```vue
<style lang="scss"></style>
```

you can run:

`npm i sass -save-dev`

## less

if youn want to use less, please run command:

`npm i less -save-dev`
